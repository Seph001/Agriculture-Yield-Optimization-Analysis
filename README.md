# Yield Optimization Analysis Dashboard

## Overview
This project is an agricultural analytics and yield optimization solution developed in Power BI to monitor crop productivity, evaluate farm performance, analyze irrigation efficiency, and assess the relationship between rainfall and agricultural yield.

The dashboard transforms agricultural and environmental data into actionable insights that support data-driven farming decisions, resource optimization, and productivity improvement across multiple geographical zones.

The analysis covers yield trends, crop performance, irrigation systems, rainfall impact, farm categories, and regional agricultural output from January 2020 to April 2026.

---

# Business Problem
Agricultural stakeholders often face challenges such as:

- Low crop productivity
- Inefficient irrigation methods
- Poor visibility into farm performance
- Inconsistent rainfall impact on yield
- Limited insight into geographical yield differences
- Difficulty optimizing agricultural resources

This project was designed to provide a centralized agricultural intelligence dashboard that enables better monitoring, planning, and optimization of farming operations.

---

The dashboard was developed to:

- Monitor agricultural yield performance

# Key Insights

## Agricultural Yield Insights
- Total agricultural yield exceeded 10.2 million kg
- Grain crops generated the highest overall output
- Plateau, Oyo, and Kano recorded strong yield performance
- Rainfall remained a major factor influencing crop productivity

## Irrigation Insights
- Sprinkler irrigation demonstrated strong yield efficiency
- Rain-fed systems showed inconsistent productivity levels
- Drip irrigation maintained balanced performance across crop categories

## Regional Insights
- North Central region achieved the highest agricultural yield
- Significant performance variation existed across states
- Regional climate and soil conditions impacted productivity outcomes
